# Email Sequences (All Products) — PRD

## Overview & Business Goal

Build comprehensive transactional and marketing email flows for all four product lines: Second Spring, Family Gift Studio (FGS), ShineOn Jewelry, and Polsia/InsightProfit. Email is the highest-ROI channel in digital marketing ($36 for every $1 spent average industry ROI) and is fully owned media — no algorithm dependency.

**Primary Goal:** Automated email sequences that (a) deliver on purchase promises, (b) convert leads to buyers, (c) ascend buyers to higher-ticket products, and (d) reactivate dormant contacts.

**Platform:** GoHighLevel (GHL) for all email automation and CRM.

---

## Revenue Model — Email Contribution

| Email Type | Revenue Mechanism | Estimated Lift |
|---|---|---|
| Welcome Sequence | Lead → first purchase conversion | 15–25% of new subscribers |
| Post-Purchase | Upsell → higher-ticket product | 10–20% upsell rate |
| Abandoned Cart | Recover lost sales | 10–15% recovery rate |
| Nurture (weekly) | Ongoing relationship → repeat purchase | 5–10% monthly buyer rate |
| Re-engagement | Recover dormant subscribers | 5–8% reactivation |
| Launch/Promotion | Revenue spikes for new products | 20–40% of launch revenue |

## Implementation Plan

### Phase 1 — Core Sequences Live (This Week, Days 1–7)

Priority order (highest revenue impact first):

1. **Second Spring Lead Magnet Welcome (1A)** — Immediate: needed when lead magnet launches
2. **ShineOn Post-Purchase (3A)** — Needed when first ShineOn sale occurs
3. **FGS Post-Purchase (2B)** — Order confirmation critical for trust
4. **Second Spring Abandoned Cart (1D)** — Recover revenue from day 1

For each sequence:
- PopeBot writes all email copy
- Human sets up GHL automation workflow + triggers
- Human configures from name, email (use custom domain, not Gmail)
- Human tests with seed email before going live

### Phase 2 — Full Suite Live (Next 2 Weeks, Days 8–21)

5. SS Post-Purchase Upsell (1B)
6. FGS Welcome Sequence (2A)
7. ShineOn Review Request + Win-Back (3B, 3C)
8. InsightProfit Welcome (4A)
9. Mother's Day Campaign sequences (all brands)
10. Weekly newsletters active for Second Spring + InsightProfit

### Phase 3 — Optimization & Scale (Month 2+)

1. A/B test subject lines (test 2 variants on first 200 sends)
2. Optimize open rates: aim for 30%+ open rate, 3%+ CTR
3. Build advanced behavioral segmentation (clicked but didn't buy → specific re-engagement)
4. Add SMS touchpoints for highest-value sequences (GHL supports SMS)
5. Launch referral email sequence (refer-a-friend program)
6. Quarterly email health audit: clean unengaged contacts, verify deliverability

## AI Execution Notes

### PopeBot Can Execute Autonomously
- Write ALL email copy for all sequences listed above (full email bodies, not just outlines)
- Write subject line variations (A/B test pairs) for each email
- Create GHL automation workflow documentation (step-by-step setup guide)
- Write trigger configuration instructions for each automation
- Generate email deliverability checklist
- Draft Mother's Day and seasonal campaign copy for all brands
- Write list segmentation rules document
- Create GHL contact tag taxonomy (naming conventions)

### Requires Human Action
- Set up GHL automations, triggers, and workflows (requires account access)
- Configure sending domain (custom domain email for deliverability)
- Connect Stripe/ShineOn webhooks to GHL
- Set up DKIM/SPF/DMARC records for sending domain
- Test all automation triggers before going live
- Approve copy before sequences go live
- Monitor deliverability (bounce rates, spam complaints) weekly

### GHL Technical Notes
- Use GHL Workflow builder for all automations (not legacy Campaigns)
- Configure "From" email as custom domain (e.g., hello@secondspring.com) — improves deliverability significantly
- Set sending limits: max 200 emails/hour per domain to warm up new domains
- Enable GHL's email bounce handling to auto-unsubscribe hard bounces
- Use "Email Statistics" in GHL to monitor per-campaign open/click rates
