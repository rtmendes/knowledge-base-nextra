# POD Design AI Agent (600 Products) — PRD

## Overview & Business Goal

Build an AI-powered pipeline that autonomously generates, uploads, and lists 600 print-on-demand products across Printify and ShineOn. The agent handles niche research → design generation → product creation → listing copy → publishing in a near-zero-human-touch workflow.

**Primary Goal:** $1,000/day in POD revenue from volume — 600 products across multiple niches, platforms, and product types, generating a long-tail of daily sales.

**The Math:** 600 products × avg $2/day/product × 80% at target velocity = ~$1,000/day. Even at $0.50/day per product average, 600 SKUs = $300/day baseline.

**Why Volume POD Works:** Most Etsy/Printify/ShineOn sellers have 10–50 products. 600 products means 600 chances to rank, 600 SEO targets, 600 social posts. AI makes this economically viable by eliminating design labor cost.

## Implementation Plan

### Phase 1 — Pipeline Build & First 50 Products (This Week, Days 1–7)

**Goal:** Working pipeline producing 50 products published by Day 7.

1. **Set Up Core Infrastructure**
   - Confirm Printify account + connect to Etsy store
   - Confirm ShineOn account + Etsy connection
   - Set up n8n or Make.com workflow automation account
   - Install/configure image generation API access (Ideogram API recommended for POD — no commercial restrictions)

2. **Build Niche Research Module**
   - Scrape Etsy bestsellers in top 10 niches (eRank or Alura tool)
   - Extract top 50 keyword phrases per niche
   - Store in structured Google Sheet or Airtable (PopeBot-readable)
   - Priority niches for Week 1: Motivational, Dog Lover, Nurse/Teacher

3. **Design Generation Workflow (50 products)**
   - Create prompt templates per niche (PopeBot generates prompt library)
   - Generate 5 design variations per niche × 10 niches = 50 designs
   - QA each design: check text legibility, print safety, no copyright issues
   - Export in correct format: PNG, 300dpi, correct dimensions per product type

4. **Printify Bulk Upload**
   - Use Printify API to create products programmatically
   - Product types for Week 1: Classic T-shirt, Mug, Tote bag
   - Set pricing: 2.5–3x base cost
   - Enable all size variants for apparel

5. **Listing Copy Generation (PopeBot)**
   - Title: keyword + emotional hook + product type (max 140 chars)
   - Description: 150–200 words, benefit-led, includes all keywords
   - Tags: 13 tags per listing (Etsy max), drawn from niche keyword research
   - Generate for all 50 products in one batch

6. **Publish to Etsy**
   - Upload via Printify's Etsy integration (auto-sync)
   - QA check: 5 random products for accuracy
   - Confirm shipping profiles set

**Success Condition for Phase 1:** 50 products live on Etsy, pipeline documented.

## Technical Specifications

### Image Generation Requirements
- Format: PNG
- Resolution: 300 DPI minimum
- T-shirt: 4500 × 5400 px (DTG) or 15 × 18 inches
- Mug: 3300 × 2550 px (wrap-around) or 2400 × 1000 px (front only)
- Jewelry (ShineOn): Per ShineOn template specifications
- Color mode: RGB for digital QA, CMYK check for print accuracy

### Printify API Integration
- Endpoint: `https://api.printify.com/v1/`
- Required: API key (from Printify account settings)
- Create product: `POST /shops/{shop_id}/products.json`
- Publish: `POST /shops/{shop_id}/products/{product_id}/publishing_succeeded.json`

### ShineOn API Integration
- Use ShineOn's official API or Zapier integration
- Products require template selection + artwork upload
- Fulfillment auto-triggered on order

## AI Execution Notes

### PopeBot Can Execute Autonomously
- Generate entire niche keyword research lists (per niche)
- Write image generation prompts for all 600 designs (batch by niche)
- Write all 600 product titles, descriptions, and tags
- Build n8n/Make.com workflow documentation
- Generate Pinterest pin descriptions from listing copy
- Draft Printify API call scripts (Python or Node.js)
- Monitor and flag underperforming products from performance data
- Generate seasonal product ideas based on calendar

### Requires Human Action
- Create Printify and ShineOn API keys (account access)
- Approve image generation API subscription and costs
- QA review of AI-generated designs (copyright/print safety check)
- Configure n8n/Make.com workflows (technical setup)
- Approve and connect Etsy store
- Final approval before each bulk publish batch
- Budget decisions for paid ads

### Cost Estimates (Pipeline Running Costs)
- Image generation API: ~$0.02–$0.05 per image × 600 = ~$15–$30 total
- n8n/Make.com: $20–$50/month
- Printify: Free to list, pay per order
- Etsy listing fee: $0.20 per listing × 600 = $120 one-time
- **Total pipeline build cost: ~$200–$300**
