# ⚡ QUICK START - 3 Steps

## Step 1: Install (2 minutes)
1. Open Chrome → `chrome://extensions/`
2. Turn ON "Developer mode" (top-right toggle)
3. Click "Load unpacked"
4. Select the `genspark-extractor-extension` folder
5. ✅ Done! You'll see a green circle icon

## Step 2: Prepare (5 minutes)
1. Go to: https://www.genspark.ai/me
2. Scroll ALL the way down to load all 911 chats
3. Scroll back to top

## Step 3: Capture (20-40 minutes)
1. **Middle-click** each chat (opens in new tab without switching)
2. Watch for green "✅ Captured #X" notification
3. Continue through all chats
4. Click extension icon → "Download CSV"
5. 🎉 Done!

---

## 🚀 Pro Tips

**Fastest method**: Middle-click (scroll wheel button)
- Opens tabs in background
- Stay on history page
- Much faster than regular clicking

**Track progress**: 
- Extension icon shows badge count
- Click icon to see detailed progress

**Take breaks**:
- Data is saved automatically
- Close Chrome anytime, progress persists
- Resume where you left off
